actualizarBotonFavorito(tarjeta) → {void}

Actualiza el estado del botón de favorito de una tarjeta. Si el videojuego no puede marcarse como favorito, el botón queda deshabilitado y el atributo data-favorito de la tarjeta se establece en "false".
